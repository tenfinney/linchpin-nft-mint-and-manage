About

Project linchpin offers a NFT minting application. It provides an easy-to-use interface which allows the user to mint an
non-fungible-token and its metadata in both IPFS content nodes and the public Ethrereum blockchain.

IPFS to store the file and its metadata, and the content idenntifier (CID) of the stored file is simultaneously recorded when token minting occurs. The metadata is also stored directly on the token and will exist as long as the Ethereum network.
